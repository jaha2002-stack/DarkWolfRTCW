DarkWolf RTCW DXR TrueRT M2/M3 Final presets

Recommended order:

1) RUN_RESET_SAFE.bat
   Reset archived cvars and launch safe non-DXR.

2) RUN_PLAY_FINAL_HIGH_QUALITY.bat
   Main mode. Uses visible RT/specular/contact shadow/temporal denoise without full-res dangerous dispatch.

3) RUN_PLAY_FINAL_STABLE.bat
   Use if HIGH QUALITY is too heavy.

4) RUN_PLAY_FINAL_EXPERIMENTAL_DXR_ULTRA.bat
   Strongest visible RT mode. It is not the main mode. Use only if HIGH QUALITY is stable.

If any mode misbehaves, run RUN_RESET_SAFE.bat, close the game, and return to STABLE or HIGH QUALITY.
